# 🌱 Agri360

**Agri360** is a smart agriculture management platform designed to help farmers, agribusinesses, and agricultural organizations make data-driven decisions. The platform provides a comprehensive 360° view of farming operations, including crop monitoring, weather insights, farm analytics, resource management, and productivity tracking.

## 🚀 Overview

Agriculture faces challenges such as unpredictable weather, resource optimization, crop diseases, and yield management. Agri360 addresses these challenges by providing a centralized digital platform that enables users to monitor, analyze, and manage agricultural activities efficiently.

## ⚙️ Installation

### Prerequisites

* Node.js (v18+)
* npm or yarn
* Supabase Account

### Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/agri360.git
```

2. Navigate to the project directory

```bash
cd agri360
```

3. Install dependencies

```bash
npm install
```

4. Configure environment variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_WEATHER_API_KEY=your_weather_api_key
```

5. Start the development server

```bash
npm run dev
```
